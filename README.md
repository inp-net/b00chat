## Overlay OBS

1. Aller sur `/?overlay`
2. Mettre ça en CSS personnalisé:

```css
body {
	background: linear-gradient(to top, rgba(0, 0, 0, 0.75), rgba(0, 0, 0, 0)) !important;
	margin: 30px !important;
	overflow: hidden;
	font-size: 48px !important;
	color-scheme: dark !important;
}
```
